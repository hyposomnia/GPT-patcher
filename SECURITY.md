# Security Policy

## Supported versions

Security fixes are currently applied to the latest released version of GPT-patcher.

## Reporting a vulnerability

Please do not open a public issue for vulnerabilities that could cause arbitrary code execution,
unsafe app-bundle replacement, backup corruption, or unintended credential exposure.

Use GitHub's **Report a vulnerability** flow for this repository. Include:

- affected GPT-patcher version and commit;
- ChatGPT Desktop version and build;
- bundled app-server version;
- reproduction steps and expected impact;
- any suggested mitigation.

Compatibility failures that stop safely without modifying unknown files can be reported as normal
GitHub issues.
